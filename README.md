# LucidVR Website

This repository contains the static website for **LucidVR**, including the public landing page and privacy policy.

The site exists solely to satisfy product, platform, and compliance requirements (e.g. store listings, analytics disclosure) and is not intended to be a general-purpose website or an open-source project.

---

## Contents

- `index.html` — One-page landing site for LucidVR
- `privacy.html` — Privacy policy covering Unity Analytics usage

---

## Deployment

The site is deployed as a static website via a hosting provider (e.g. GitHub Pages, Cloudflare Pages, or equivalent).

No build step is required. Files are served as-is.
